Files:
	zero_one_knap.c contains program of 0/1 knapsack problem using Dynaic programming approach.
	input_knap.txt contains input
		first line contain weight of knapsck
		second line contains totle items
		next three line contain profit
		next three line contain weight of items
	out.txt contains output of the program:
		first line contain totle profit
		second line contains output vector
		third line contains index of items selected
		
commands:
	>gcc zero_one_knap.c
	>./a.out input_knap.txt out.txt
